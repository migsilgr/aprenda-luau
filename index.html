-- fundamentos do luau

Os pilares do
Luau no Roblox

📦

Variáveis

Guardam informações como vida do jogador, velocidade ou pontuação. Em Luau usamos local para declarar.

🔀

Condicionais

Tomada de decisões: se o jogador tiver 0 de vida, fim de jogo. A base de toda lógica de gameplay.

🔁

Laços (Loops)

Repita ações — como checar colisões a cada frame ou gerar inimigos em intervalos com while e for.

🧩

Funções

Blocos reutilizáveis de lógica. Crie uma função DarDano() e chame-a sempre que precisar.

🎮

Eventos (Events)

Luau é orientado a eventos: Touched, ClickDetector, PlayerAdded — reaja ao que acontece no jogo.

🏗️

Services do Roblox

APIs nativas como Players, Workspace e ReplicatedStorage — acessadas via game:GetService().

Variáveis e tipos

Em Luau, toda variável local começa com local. Você pode guardar números, textos, booleanos e até objetos do jogo.

  • Sempre use local — variáveis globais são lentas e perigosas
  • Strings usam aspas duplas "texto" ou simples 'texto'
  • Concatenação de texto usa .. em vez de +
Luau 
-- Declarando variáveis locais
local nomeJogador = "PlayerOne"
local vida        = 100
local estaVivo    = true

-- Concatenação com ..
local msg = "Jogador: " .. nomeJogador
           .. " | Vida: " .. vida

print(msg)
-- → Jogador: PlayerOne | Vida: 100

Condicionais + Eventos

Detecte quando uma peça é tocada com o evento Touched e use if/elseif/else para decidir o que fazer.

  • Luau usa elseif (tudo junto, sem espaço)
  • Blocos terminam com end
  • :Connect() registra a função no evento
Luau 
local parte = script.Parent
local vida  = 100

parte.Touched:Connect(function(hit)
  local personagem = hit.Parent
  local humanoide =
    personagem:FindFirstChild("Humanoid")

  if humanoide then
    vida = vida - 10
    print("Vida restante: " .. vida)
  end
end)

Funções e loops

Crie funções nomeadas e use for numérico para repetir ações — por exemplo, gerar vários objetos no mapa.

  • for i = 1, 5 do repete 5 vezes
  • task.wait() é o jeito moderno de pausar (não use wait())
  • Use Instance.new() para criar objetos
Luau 
local Workspace = game:GetService("Workspace")

local function criarMoeda(posX)
  local moeda = Instance.new("Part")
  moeda.Size     = Vector3.new(1, 1, 1)
  moeda.Position = Vector3.new(posX, 5, 0)
  moeda.BrickColor = BrickColor.new("Bright yellow")
  moeda.Parent = Workspace
end

-- Cria 5 moedas lado a lado
for i = 1, 5 do
  criarMoeda(i * 3)
  task.wait(0.2)
end
